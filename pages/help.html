How to style a note?

  • To edit an <h1> use the selector .note h1
  • To edit an <H2> use the selector .note h2
  • To edit an <H3> use the selector .note h3
  • To edit a <P> use the selector .note p
  • To edit an <UL> use the selector .note ul
  • To edit a <LINK> use the selector .note a
  • To edit an <IMG> use the selector .note img

How to style a note card?

  • To edit an <H1>use the selector .card h1
  • To edit an <H2> use the selector .card h2
  • To edit an <H3> use the selector .card h3
  • To edit a <P> use the selector .card p
  • To edit an <UL> use the selector .card ul
  • To edit a <LINK> use the selector .card h2 span
  • To edit an <IMG> use the selector .card img
